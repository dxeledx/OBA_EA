# 2025-12-23 — OEA / OEA-ZO Ablations (LOSO, BNCI2014_001, 4-class)

This note repeats the previous OEA/OEA-ZO experiment set under a **4-class** MI setting:
`left_hand`, `right_hand`, `feet`, `tongue`.

## Common setup
- Dataset: MOABB `BNCI2014_001` (BCI Competition IV 2a)
- Sessions: `0train`
- Preprocess: either `moabb` or `paper_fir` (causal FIR, order=50, Hamming)
- Bandpass: 8–30 Hz, resample: 250 Hz
- Epoch window: `tmin=0.5s`, `tmax=3.5s`
- Evaluation: LOSO (9 subjects)
- Metrics: mean accuracy shown below; full metrics per subject are in each run folder

## Key observations
- **4-class + moabb preprocess**: `oea-*` / `oea-zo-*` are **worse than EA** in this run.
- **4-class + paper_fir**: `oea-*` / `oea-zo-*` are closer to EA but still generally **below EA** under these tested configs.
- The high-variance regime (`q_blend=0.5` + aggressive ZO) is **consistently worse than EA**, similar to the 2-class probe.
